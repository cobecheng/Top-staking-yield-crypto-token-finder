# Result Files

The pipeline writes CSV files locally. They are ignored by Git and are not committed to the public repository.

## Results Guide

| File | Generated By | Use It For |
| --- | --- | --- |
| `top_1000_marketcap_coingecko_canonical.csv` | `fetch_top1000_marketcap_coingecko_canonical.py` | Canonical top-1000 token universe ranked by market cap. |
| `top_1000_official_links_from_coingecko.csv` | `enrich_official_links_from_canonical_coingecko.py` | First-pass official websites and X handles from CoinGecko. |
| `top_1000_official_links_validated.csv` | `validate_official_links.py` | Validated websites, official X candidates, and validation status. |
| `top_1000_staking_baseline.csv` | `prefill_staking_from_stakingrewards.py` | Staking Rewards baseline matches and any discovered APY/APR fields. |
| `top_1000_staking_baseline_stakingrewards_only.csv` | `prefill_staking_from_stakingrewards.py` | Summary subset of rows matched to Staking Rewards. |
| `top_1000_staking_unresolved_queue.csv` | `build_unresolved_staking_queue.py` | Tokens still missing baseline yield and queued for website/X discovery. |
| `top_1000_staking_website_unresolved.csv` | `discover_staking_from_websites.py` | Website-side staking pages, yield snippets, and discovery notes for unresolved tokens. |
| `top_1000_x_staking_signals_unresolved_resumed.csv` | `scan_x_staking_signals_batch.mjs` | Official-X staking-signal findings with source post URLs and selected links. |
| `top_1000_x_search_fallback.csv` | `scan_x_search_staking_fallback.mjs` | Broader X search fallback for rows still unresolved after official-X scanning. |

The X scripts also have smaller default output names such as `top_100_x_staking_signals.csv` and `top_100_x_search_fallback.csv`. Pass `--output` when you want the filenames above.

## Expected Columns

### `top_1000_marketcap_coingecko_canonical.csv`

`rank`, `name`, `symbol`, `coingecko_id`, `market_cap_usd`, `current_price_usd`, `fully_diluted_valuation_usd`, `circulating_supply`, `total_supply`, `max_supply`, `last_updated`

### `top_1000_official_links_from_coingecko.csv`

Generated as an intermediate file before validation. Columns include token identity, CoinGecko official website fields, X handle fields, and CoinGecko metadata.

### `top_1000_official_links_validated.csv`

`rank`, `name`, `symbol`, `coingecko_id`, `market_cap_usd`, `coingecko_website`, `coingecko_x`, `website_final_url`, `website_final_domain`, `website_http_code`, `website_content_type`, `x_found_on_website`, `selected_official_x`, `selected_official_x_source`, `selected_official_x_score`, `validation_status`, `validation_notes`

### `top_1000_staking_baseline.csv`

`rank`, `name`, `symbol`, `coingecko_id`, `market_cap_usd`, `current_price_usd`, `fully_diluted_valuation_usd`, `circulating_supply`, `total_supply`, `max_supply`, `last_updated`, `baseline_source`, `baseline_asset_url`, `baseline_staking_link`, `baseline_yield_pct`, `baseline_yield_type`, `baseline_match_method`, `needs_website_scan`, `needs_x_scan`

### `top_1000_staking_unresolved_queue.csv`

`rank`, `name`, `symbol`, `coingecko_id`, `market_cap_usd`, `current_price_usd`, `fully_diluted_valuation_usd`, `circulating_supply`, `total_supply`, `max_supply`, `last_updated`, `baseline_source`, `baseline_asset_url`, `baseline_staking_link`, `baseline_yield_pct`, `baseline_yield_type`, `baseline_match_method`, `needs_website_scan`, `needs_x_scan`, `website_final_url`, `website_http_code`, `selected_official_x`, `validation_status`

### `top_1000_staking_website_unresolved.csv`

`rank`, `name`, `symbol`, `coingecko_id`, `website_final_url`, `selected_official_x`, `staking_status`, `home_page_keywords`, `candidate_staking_pages`, `selected_staking_page`, `selected_page_title`, `selected_page_http_code`, `yield_pct`, `yield_type`, `yield_snippet`, `discovery_source`, `discovery_notes`

### `top_1000_x_staking_signals_unresolved_resumed.csv`

`rank`, `name`, `symbol`, `coingecko_id`, `x_username`, `profile_url`, `scan_status`, `posts_returned`, `staking_signal_count`, `latest_signal_post_url`, `latest_signal_post_date`, `latest_signal_post_text`, `latest_signal_keywords`, `selected_link`, `link_source`, `reply_link_count`

By default, `latest_signal_post_text` is redacted as `[redacted: see post URL]`. Use `--include-post-text` only for private local analysis.

### `top_1000_x_search_fallback.csv`

`rank`, `name`, `symbol`, `coingecko_id`, `query`, `search_status`, `match_count`, `top_post_url`, `top_post_author`, `top_post_date`, `top_post_text`, `top_post_keywords`

By default, `top_post_text` is redacted as `[redacted: see post URL]`. Use `--include-post-text` only for private local analysis.
