# Security

This repository is designed to run as a local research workflow. It should not
contain production credentials, private browser state, or exchange account data.

## What Not To Commit

- `.env` files or API keys
- X/Twitter session cookies or browser profiles
- wallet seed phrases, private keys, or exchange credentials
- local cache files generated by the crawlers
- raw social-media post dumps beyond the minimal redacted CSV fields

## Reporting

If you find a sensitive file, leaked credential, or unsafe default, open a
private report to the maintainer instead of posting the value in a public issue.
